# 🧮 Calculadora Android

Aplicación de calculadora desarrollada en **Android Studio utilizando Kotlin**.  
La app permite realizar operaciones matemáticas básicas con una interfaz moderna y sencilla.

---

## 📱 Funcionalidades

- ➕ Suma
- ➖ Resta
- ✖️ Multiplicación
- ➗ División
- Validación de entrada de números
- Visualización del resultado en pantalla
- Interfaz moderna estilo calculadora

---

## 🛠️ Tecnologías utilizadas

- **Kotlin**
- **Android Studio**
- **XML Layouts**
- **Material Design**

---

## 📂 Estructura del proyecto

Calculadora
│
├── MainActivity.kt → Lógica de la calculadora
├── activity_main.xml → Diseño de la interfaz
└── AndroidManifest.xml → Configuración de la aplicación
---

## 🚀 Cómo ejecutar el proyecto

1. Clonar el repositorio
2. Abrir el proyecto en **Android Studio**
3. Ejecutar la aplicación en un **emulador o dispositivo físico**
